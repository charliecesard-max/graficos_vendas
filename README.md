# Visualização de Dados de Vendas com Matplotlib

Este projeto demonstra o uso básico da biblioteca Matplotlib em Python para criar diferentes tipos de gráficos.  
São exibidas informações fictícias sobre vendas, estoque e categorias de produtos.

---

## Requisitos

- Python 3.8 ou superior
- Biblioteca Matplotlib

### Instalação

Antes de executar o código, instale o Matplotlib (caso ainda não tenha):

```bash
pip install matplotlib
